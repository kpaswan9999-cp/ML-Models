# 🤖 ML-Models Repository
This repository contains multiple Machine Learning projects and model implementations developed using Python and Jupyter Notebook. The goal of this repository is to practice, understand, and demonstrate different supervised learning algorithms applied to real-world datasets.

# Overview

The projects included in this repository cover fundamental and practical Machine Learning concepts such as:

  Data preprocessing.
  Exploratory Data Analysis (EDA).
  Model training & evaluation.
  Performance comparison.
  Real-world dataset applications.
  Each notebook explains the workflow step-by-step, making it useful for learning and reference purposes.

#  📂 Projects Included

## 🔹1.Linear Regression – Climate Analysis

    Built a Linear Regression model.
    Studied relationships between climate variables. 
    Demonstrates regression modeling and evaluation metrics.

## 🔹2.Logistic Regression – Social Network Ads

     Classification model predicting user purchase behavior.
     Applied Logistic Regression for binary classification.
     Includes feature scaling and accuracy evaluation.

## 🔹3.KNN – Global Greenhouse Gas Emissions

      Implemented K-Nearest Neighbors (KNN) algorithm.
      Analysis of greenhouse gas emission patterns.
      Data visualization and prediction modeling.

## 🔹4.Naive Bayes – Hotel Booking Prediction
      
      Implemented Naive Bayes classifier.
      Predicts booking outcomes using customer data.
      Focus on probabilistic learning methods.

## 🔹5.Support Vector Classifier – EV Battery Degradation

      Used Support Vector Machine (SVM/SVC).
      Predicts electric vehicle battery degradation.
      Demonstrates advanced classification techniques.
